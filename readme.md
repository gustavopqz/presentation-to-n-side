# Presentation to N-SIDE

A personal portfolio and story built as a React SPA for N-SIDE.

## Live deployment

The site is deployed on Netlify at:

https://gustavo-n-side.netlify.app/

## Why this project exists

This project is a narrative about who I am as an engineer and why I want to join N-SIDE.

- **N-SIDE as a destination:** I chose this company because it is recognized in Belgium's tech ecosystem for strong life sciences impact, innovation, and thoughtful engineering.
- **My story:** I built this site to show my history, my career, and my values in a way that a resume alone cannot.
- **Why N-SIDE:** I want to work somewhere that improves healthcare outcomes, supports growth, and surrounds me with smart, diverse people.

## What this project shows N-SIDE about me

I come with a grounded engineering path, built in Brazil and now continuing in Europe. This site highlights:

- the people and purpose behind my work,
- the way I think about long-term impact,
- why I moved from survival-driven decisions to growth-driven opportunities.

### 3. My values and fit

I want to be part of a team that:

- values learning and improvement,
- rewards initiative,
- supports a flexible, creative environment,
- brings together diverse technical and scientific expertise.

## Why React here

This project is also a practical experiment in React, built specifically to contrast with the Angular experience I already know.

### React vs Angular for this project

- **React is component-focused**: the app is composed from small UI pieces and route pages.
- **React is unopinionated**: there is freedom in how state, routing, and page layout are structured.
- **Angular is more convention-driven**: React lets this presentation feel more lightweight and flexible while still being typed with TypeScript.

That makes this project a great place to show both my ability to learn a new stack and my ability to compare frameworks from experience.

## Creative presentation goals

This isn't a generic portfolio. It is designed to feel like a written, thoughtful conversation:

- the homepage tells a personal story,
- the work page shows projects and principles,
- the team page explains how I contribute to culture,
- the N-SIDE page explains why the company is the right next step.

## Technologies used

- `React` with `TypeScript`
- `Vite` for fast development and static builds
- `@tanstack/react-router` for SPA routing
- `@tanstack/react-query` for data/query structure support
- `Tailwind CSS` for styling utilities
- `Framer Motion` for animated transitions
- `Lucide React` for iconography

## Build and run

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

### Static production build

```bash
npm run build
```

### Preview the build

```bash
npm run preview
```
